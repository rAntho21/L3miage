Règle de jeu

Le jeu du pendu consiste à trouver un mot généré aléatoirement en essayant de trouver les bonnes lettres une par une.
Vous aurez un certain nombre de chances pour trouver le mot.
Lorsque vous pensez avoir le mot, vous pouvez tenter votre chance, si vous échouez, alors vous perdez une vie.
